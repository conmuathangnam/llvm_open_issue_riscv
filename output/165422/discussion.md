### Comment 2 - davemgreen

The code is expecting there to be at least one function in the module. It works if a dummy function is added. https://godbolt.org/z/cdbWbGKa8. I would guess that from C++ the static constructor list usually contains the definition of the function.

The code in ARMAsmPrinter::emitXXStructor needs to look somewhere other than Subtarget->isTargetELF() for whether the target is elf.
