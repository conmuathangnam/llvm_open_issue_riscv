### Comment 2 - keinflue

Crashes on invalid because the builtin `__builtin_goacc_parlevel_size` is not supported.

Reduces further to:
```cpp
enum E {
  A,
  B = __builtin_UNKNOWN_BUILTIN(UNKNOWN_IDENTIFIER),
  C = (A | B),
};
```
where `__builtin_UNKNOWN_BUILTIN` is an invalid builtin and `UNKNOWN_IDENTIFIER` any undeclared identifier.

Occurs only in C mode, not in C++ mode.

Crash since Clang 21: https://godbolt.org/z/P8Yc8xnhq
