# [opt] [ctx-instr-lower] Assertion Error When GUID Metadata Is Unavailable

**Author:** fraglantia
**URL:** https://github.com/llvm/llvm-project/issues/166302
**Status:** Open
**Labels:** PGO, crash

## Body

`ctx-instr-lower` lowers tags generated by `ctx-instr-gen` to executable IR code but breaks if GUID is unavailable (e.g., from `assign-guid`).

Reproducer:
https://godbolt.org/z/jPPrn7zj6
